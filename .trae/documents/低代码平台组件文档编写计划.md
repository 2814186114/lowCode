# 低代码平台基础组件文档编写计划

## 1. 文档格式要求

### 1.1 遵循Ant Design文档格式
每个组件文档将包含以下核心部分：
- **组件名称**：中英文名称
- **简介**：组件的核心功能描述
- **何时使用**：组件适用场景
- **代码演示**：不同使用方式的代码示例
- **API**：完整的属性、事件、方法列表
- **注意事项**：使用时的注意事项

### 1.2 文档目录结构
```
/docs
  /widgets           # 基础组件文档目录
    Button.md        # 按钮组件文档
    Input.md         # 输入框组件文档
    Select.md        # 下拉选择组件文档
    Switch.md        # 开关组件文档
    Slider.md        # 滑块组件文档
    DatePicker.md    # 日期选择组件文档
    Row.md           # 行布局组件文档
    Col.md           # 列布局组件文档
    LoginForm.md     # 登录表单组件文档
  README.md          # 文档首页
```

## 2. 文档内容规范

## 4. 文档示例

### Button组件文档示例
```markdown
# Button 按钮

按钮用于开始一个即时操作。

## 何时使用

- 响应用户点击行为，触发相应的业务逻辑。
- 支持多种样式和状态，适应不同场景需求。

## 代码演示

### 基本用法

```jsx
import { Button } from '@ant-design/react-lowcode';

function App() {
  return (
    <div>
      <Button type="primary">主按钮</Button>
      <Button>默认按钮</Button>
      <Button type="dashed">虚线按钮</Button>
      <Button type="text">文本按钮</Button>
      <Button type="link">链接按钮</Button>
    </div>
  );
}
```

### 按钮状态

```jsx
import { Button } from '@ant-design/react-lowcode';

function App() {
  return (
    <div>
      <Button type="primary" loading>加载中</Button>
      <Button type="primary" disabled>禁用</Button>
      <Button danger>危险按钮</Button>
      <Button type="primary" danger>危险主按钮</Button>
    </div>
  );
}
```

## API

### 属性

| 属性名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| type | string | 'default' | 按钮类型，可选值为 `primary` `dashed` `link` `text` `default` |
| size | string | 'middle' | 按钮大小，可选值为 `large` `middle` `small` |
| danger | boolean | false | 是否为危险按钮 |
| loading | boolean | false | 是否处于加载状态 |
| disabled | boolean | false | 是否禁用 |
| onClick | function | - | 点击事件回调，参数为 `event` |
| children | ReactNode | - | 按钮内容 |

### 事件

| 事件名 | 说明 | 回调参数 |
|--------|------|----------|
| onClick | 点击按钮时触发 | event: MouseEvent |

### 方法

| 方法名 | 说明 | 参数 | 返回值 |
|--------|------|------|--------|
| focus | 获取焦点 | - | void |
| blur | 失去焦点 | - | void |

## 注意事项

- 按钮的 `onClick` 事件会冒泡，如果不希望冒泡，请在事件处理函数中调用 `event.stopPropagation()`。
- 当按钮内容为空时，按钮将显示为默认文本。
